# Portfolio Visual Export Guide

A comprehensive guide to exporting visuals for Marco's portfolio, organized by case study with specific export specs, Figma sources, and prioritization.

---

## Priority Tiers

| Tier | Description | Action |
|------|-------------|--------|
| **P0** | Hero shots, must-have for case study credibility | Export before applying |
| **P1** | Key decision visuals that support the narrative | Export for strong applications |
| **P2** | Process/gallery shots that add depth | Export as time permits |
| **P3** | Nice-to-have polish | Can remain placeholders initially |

---

## Export Specifications

### Standard Dimensions by Aspect Ratio

| Aspect Ratio | Recommended Size | Use Case |
|--------------|------------------|----------|
| 16:9 | 1920x1080px or 2880x1620px (2x) | Desktop UI, dashboards, before/after |
| 3:2 | 1800x1200px or 2700x1800px (2x) | Detailed UI sections |
| 3:4 | 1200x1600px or 1800x2400px (2x) | Mobile screens (portrait) |
| 9:16 | 1080x1920px or 1620x2880px (2x) | Mobile screens (tall) |

### Export Format
- **UI Screenshots**: PNG with 2x resolution
- **Mockups/Composites**: PNG or JPG (90% quality)
- **Diagrams**: PNG with transparent background option

### Annotation Tips
- Use consistent annotation style (arrows, callouts)
- Highlight key interactions or decision points
- Consider before/after framing for impact
- Device frames optional but add polish

## Export Workflow

### Quick Export Checklist

1. **Open Figma file** at the specified node
2. **Select frame** to export
3. **Export settings**: PNG, 2x resolution
4. **File naming**: `[case-study]-[description].png` (e.g., `fb-ordering-hero.png`)
5. **Save to**: `public/images/[case-study]/`

### Annotation Workflow (if needed)

1. Duplicate the frame in Figma
2. Add annotation layer on top
3. Use consistent callout style (consider making a reusable annotation component)
4. Export the annotated version

### Mockup Workflow (for heroes)

1. Use device mockup templates (Figma has built-in ones)
2. Place exported UI into device frame
3. Consider perspective mockups for hero variety
4. Export at 2x for retina displays
