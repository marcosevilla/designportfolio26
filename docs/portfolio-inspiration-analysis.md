## Common Patterns Across Portfolios

### 1. Clear, Concise Positioning
All portfolios lead with a **single sentence** that captures:
- Who they are
- What they do
- Their unique value

**Examples:**
- Pranathi: "translates complex technical concepts into simple user experiences"
- Natalie: "building elegant software where complexity feels effortless"
- Oscar: "challenging problems with thoughtful design"

### 2. Minimal Navigation
None of these portfolios have complex menus. Typically:
- 3-5 nav items max
- Direct labels (Work, About, Contact)
- Social links prominently placed

### 3. Personality Integration
Every portfolio reveals personality:
- **Pranathi:** Spotify now-playing, hobbies
- **Marco:** Philosophy statements, tool ecosystem
- **Natalie:** "SET VIBES" customization, bold typography
- **Oscar:** "Speculative projects that demand unconventional thinking"

### 4. Visual Hierarchy Over Text
All portfolios are **scannable**:
- Large typography for key statements
- Generous whitespace
- Visual projects take center stage
- Minimal paragraph copy

### 5. Technical Credibility
All designers demonstrate technical chops:
- **Pranathi & Natalie:** Built with Next.js
- **Marco:** Complex Webflow interactions
- **Oscar:** Custom Squarespace modifications
- All show they can code or work closely with developers

### 6. Strategic Positioning
Everyone positions beyond "execution":
- Strategic thinking
- Problem-solving
- Business impact
- Systems thinking

### 7. Mobile Responsiveness
All portfolios are fully responsive with:
- Adaptive layouts
- Optimized images
- Touch-friendly interactions

## Visual Design Recommendations

### Color Palette
Based on Marco's preferences + inspirations:

**Primary:**
- Burnt orange (#E67E22 or similar) - accent/highlights
- Near-black (#1A1A1A) - primary text
- Pure white (#FFFFFF) - backgrounds
- Warm grey (#F5F5F5) - secondary backgrounds

**Supporting:**
- Mid-grey (#666666) - secondary text
- Light grey (#E0E0E0) - borders/dividers

### Typography
Inspired by portfolios analyzed:

**Option 1: Mono + Sans (like Natalie)**
- Headings: Monospace font (JetBrains Mono, Space Mono)
- Body: Clean sans-serif (Inter, Helvetica)
- Gives technical credibility

**Option 2: Modern Sans (like Pranathi/Oscar)**
- Single sans-serif family (Inter, SF Pro, Work Sans)
- Use weight variation (Light, Regular, Bold, Black)
- Clean, professional, versatile

**Recommendation:** Option 2 for broader appeal, easier readability

### Layout System
- Max content width: 1200px
- Generous margins: 60-100px on desktop
- Grid: 12-column for flexibility
- Vertical rhythm: 8px base unit
- Card padding: 32-48px
- Section spacing: 120-160px

### Interactive Elements
- Hover states: Subtle lift (4-8px) + shadow
- Transitions: 200-300ms ease-out
- Focus states: Burnt orange outline
- Smooth scrolling between sections
- No auto-playing videos/animations

### Typography Scale
- H1 (Hero): 64-80px
- H2 (Section): 48-56px
- H3 (Subsection): 32-40px
- Body Large: 20-24px
- Body: 16-18px
- Caption: 14px
