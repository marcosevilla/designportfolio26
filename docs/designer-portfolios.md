## Semplice / Independent

- **Tobias van Schneider** — Co-founder of Semplice (portfolio system built for case studies). His writing on case study design is canonical. Key principles: senior designers tell stories not essays, show personality not just process, first 2-3 seconds determine if a recruiter stays.
  - Portfolio: [vanschneider.com](https://vanschneider.com/)
  - Guide: [A Visual Guide to Writing Portfolio Case Studies](https://vanschneider.medium.com/a-visual-guide-to-writing-portfolio-case-studies-ad5dbc513f37)
